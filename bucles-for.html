🧭 Resumen del Bucle FOR

                   

                        ¿Cuándo usar FOR? Cuando sabes exactamente                         cuántas veces quieres repetir algo:                    

                   
                           
  •                             Recorrer una lista de elementos (islas, tripulantes, tesoros)                        
    •                        
  • Repetir una acción un número específico de veces
    •                        
  • Procesar cada elemento de una colección
    •                    
                                        

Sintaxis básica con listas:

                    
for elemento in lista:

    # código a repetir

    print(elemento)
                   

Sintaxis con range() para números:

                    
for numero in range(inicio, fin, paso):

    # código a repetir

    print(numero)
                                        

Ejemplos de range():

                   
                           
  • range(5) → genera números del 0 al 4 (5 números)
    •                        
  • range(1, 6) → genera números del 1 al 5
    •                        
  • range(1, 10, 2) → genera números impares del 1 al 9 (1, 3, 5, 7, 9)
    •                        
  • range(10, 0, -1) → genera números del 10 al 1 en orden descendente
    •                    
                     

                        ¡Perfecto para aventuras piratas organizadas!                         Usa listas cuando tengas elementos específicos, y range() cuando necesites números secuenciales.
